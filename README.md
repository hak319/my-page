# 이은학 포트폴리오 웹사이트

깔끔하고 현대적인 디자인의 포트폴리오 웹사이트입니다.  
HTML, CSS, JavaScript로 제작되었으며 다크모드와 반응형 디자인을 지원합니다.

## 주요 기능

- 홈, 학력, 기술, 자격증, 수상, 연락처 섹션  
- 부드러운 페이지 전환 애니메이션  
- 다크모드 토글 기능  
- 모바일 대응 반응형 레이아웃  

## 기술 스택

- HTML5  
- CSS3 (커스텀 프로퍼티, 미디어 쿼리)  
- JavaScript (DOM 조작)  
- Google Fonts (Pretendard)  

## 설치 및 실행

1. 저장소 클론 또는 다운로드  
2. `index.html` 파일을 브라우저로 열기  

혹은

https://my-page-orcin.vercel.app/#
